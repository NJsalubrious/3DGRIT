Dynamic Control Shapes

Switching control types on the fly. Building a custom tool panel to automatically swap control curves based on context, visibility needs, and animator preference.

Tools UI Blender
Primary_Signal // YouTube Watch_External ▸

Eliminating Clutter

▸

A rig with 400 controls on screen is unusable. Traditional layered visibility approaches in Blender can turn things into a mess of hunting via the Properties panel.

The solution implemented here is a fluid UI structure where selecting parent controls surfaces secondary and tertiary rings automatically, hiding the visual noise until it's specifically required by the user.

Shape Definitions

▸

Controls map dynamically to JSON libraries mapping exact WGDC offsets and rotational scale data. It guarantees parity whether an animator is working at 10x scale or base character dimensions. They can change shape types (box, rings, arrows) with immediate UI interaction, removing the standard Blender friction.
